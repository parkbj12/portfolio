# 박병관 포트폴리오

개인 포트폴리오 웹사이트입니다.

## 소개

경기과학기술대학교 자동차과와 강남대학교 인공지능학과를 거치며 공학적 사고와 데이터 분석 역량을 키워온 박병관의 포트폴리오입니다.

## 기술 스택

- React
- Vite
- Tailwind CSS

## 프로젝트 목록

1. AI Interview - AI 기반 모의면접 서비스
2. FlatFootball - 풋살 매치 예약 시스템
3. YOLO 기반 산불 조기 탐지 AI 모델

## 실행 방법

```
npm install
npm run dev
```
